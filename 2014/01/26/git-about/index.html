  1. RCS(Revision Control System) дҝ®и®ўжҺ§еҲ¶зі»з»ҹпјҢзү№зӮ№:
    1. з®ҖеҚ•
    2. дҪҝз”ЁLockжңәеҲ¶йҳІжӯўеӨҡдёӘејҖеҸ‘дәәе‘ҳеҜ№еҗҢдёҖдёӘж–Үд»¶еҗҢж—¶иҝӣиЎҢдҝ®ж”№.
  2. CVS(Cocurrent Version System)е№¶еҸ‘зүҲжң¬зі»з»ҹ,е»әз«ӢеңЁRCSеҹәзЎҖдёҠ,жңҖжөҒиЎҢзҡ„ејҖж”ҫжәҗд»Јз ҒзүҲжң¬жҺ§еҲ¶зі»з»ҹ,зү№зӮ№:

    2. жңҖеӨ§дјҳзӮ№еңЁдәҺеӨҡеҗҚејҖеҸ‘дәәе‘ҳеҸҜд»ҘеҗҢж—¶еҜ№дёҖдёӘж–Үд»¶иҝӣиЎҢдҝ®ж”№.е…Ғи®ёеҗҲе№¶.иҝҷе°ұвҖңе№¶еҸ‘вҖңејҖеҸ‘.
  3. SVN(SubVersion)
    1. зӣ®еҪ•зҡ„зүҲжң¬жҺ§еҲ¶,CVS еҸӘиғҪеҜ№ж–Үд»¶иҝӣиЎҢзүҲжң¬жҺ§еҲ¶пјҢдёҚиғҪеҜ№зӣ®еҪ•иҝӣиЎҢзүҲжң¬жҺ§еҲ¶.CVS еҸӘиғҪжіЁж„ҸеҲ°пјҢдёҖдёӘж–Үд»¶еңЁдёҖдёӘдҪҚзҪ®иў«еҲ йҷӨдәҶпјҢиҖҢеңЁдёҖдёӘж–°дҪҚзҪ®еҲӣе»әдәҶеҸҰеӨ–дёҖдёӘж–Үд»¶гҖӮз”ұдәҺе®ғдёҚдјҡиҝһжҺҘдёӨдёӘж“ҚдҪңпјҢеӣ жӯӨд№ҹеҫҲе®№жҳ“дҪҝж–Үд»¶еҺҶеҸІиҪЁиҝ№дёўеӨұ,SVNеҸҜд»Ҙ
    2. еҺҹеӯҗжҖ§жҸҗдәӨ,CVS йҮҮз”ЁзәҝжҖ§гҖҒдёІиЎҢзҡ„жү№йҮҸжҸҗдәӨпјҢеҚідҫқж¬Ўең°пјҢдёҖдёӘжҺҘдёҖдёӘең°жү§иЎҢжҸҗдәӨпјҢжҜҸжҲҗеҠҹжҸҗдәӨдёҖдёӘж–Үд»¶пјҢиҜҘж–Үд»¶зҡ„дёҖдёӘж–°зҡ„зүҲжң¬еҚіиў«и®°еҪ•еҲ°зүҲжң¬еә“дёӯпјҢжҸҗдәӨж—¶з”ЁжҲ·жҸҗдҫӣзҡ„ж—Ҙеҝ—дҝЎжҒҜиў«йҮҚеӨҚең°еӯҳеӮЁеҲ°жҜҸдёҖдёӘиў«дҝ®ж”№зҡ„ж–Үд»¶зҡ„зүҲжң¬еҺҶеҸІдёӯгҖӮCVS дёІиЎҢжү№йҮҸжҸҗдәӨжЁЎејҸзҡ„ејҠз«ҜеңЁдәҺ пјҚеҪ“д»»дҪ•еҺҹеӣ йҖ жҲҗжү№йҮҸж“ҚдҪңзҡ„дёӯж–ӯж—¶пјҲе…ёеһӢеҺҹеӣ еҢ…жӢ¬пјҡзҪ‘з»ңдёӯж–ӯгҖҒе®ўжҲ·з«Ҝжӯ»жңәзӯүпјүпјҢзүҲжң¬еә“еҫҖеҫҖеӨ„дәҺдёҖдёӘдёҚдёҖиҮҙзҡ„зҠ¶жҖҒпјҡеҺҹжң¬еә”иҜҘе…ЁйғЁе…Ҙеә“зҡ„ж–Үд»¶еҸӘжңүдёҖйғЁеҲҶе…Ҙеә“пјҢеҫҲжңүеҸҜиғҪзүҲжң¬еә“дёӯзҡ„жңҖж–°зүҲжң¬дёҚиғҪйЎәеҲ©зј–иҜ‘пјҢжӣҙдёәдёҘйҮҚзҡ„жҳҜпјҢйҡҸзқҖе…¶д»–зҡ„з”ЁжҲ·жү§иЎҢcvs update ж“ҚдҪңпјҢиҜҘдёҚдёҖиҮҙжҖ§е°Ҷиҝ…йҖҹеңЁејҖеҸ‘еӣўйҳҹдёӯжү©ж•ЈпјҢд»ҺиҖҢдёҘйҮҚеҪұе“Қеӣўйҳҹзҡ„ејҖеҸ‘ж•ҲзҺҮпјҢе№¶еӯҳеңЁиҙЁйҮҸйҡҗжӮЈгҖӮеҸҰеӨ–пјҢеҒҮеҰӮиҜҘжү№йҮҸжҸҗдәӨзҡ„дёӯж–ӯжІЎжңүиў«еҸҠж—¶еҸ‘зҺ°пјҢејҖеҸ‘еӣўйҳҹеҫҖеҫҖиҰҒиҠұжӣҙеӨҡзҡ„ж—¶й—ҙиҝӣиЎҢиҪҜд»¶и°ғиҜ•е’ҢжҺ’й”ҷгҖӮ

Git, Git жҳҜз”ЁдәҺ Linux еҶ…ж ёејҖеҸ‘зҡ„зүҲжң¬жҺ§еҲ¶е·Ҙе…·гҖӮдёҺеёёз”Ёзҡ„зүҲжң¬жҺ§еҲ¶е·Ҙе…· CVS, Subversion зӯүдёҚеҗҢпјҢе®ғйҮҮз”ЁдәҶеҲҶеёғејҸзүҲжң¬еә“зҡ„ж–№ејҸпјҢдёҚеҝ…жңҚеҠЎеҷЁз«ҜиҪҜд»¶ж”ҜжҢҒпјҢдҪҝжәҗд»Јз Ғзҡ„еҸ‘еёғе’ҢдәӨжөҒжһҒе…¶ж–№дҫҝгҖӮ Git зҡ„йҖҹеәҰеҫҲеҝ«пјҢиҝҷеҜ№дәҺиҜёеҰӮ Linux kernel иҝҷж ·зҡ„еӨ§йЎ№зӣ®жқҘиҜҙиҮӘз„¶еҫҲйҮҚиҰҒгҖӮ Git жңҖдёәеҮәиүІзҡ„жҳҜе®ғзҡ„еҗҲе№¶и·ҹиёӘпјҲmerge tracingпјүиғҪеҠӣгҖӮgitжӣҙеҠ йҖӮеҗҲеҲҶеёғејҸејҖеҸ‘йЎ№зӣ®гҖӮиҖҢsvnпјҲеҪ“з„¶е…Ёз§°жҳҜsubversionпјүеҲҷжӣҙйҖӮеҗҲдәҺйӣҶдёӯејҸеӨ§еһӢејҖеҸ‘йЎ№зӣ®гҖӮд№ҹжңүеңЁgitд№ӢдёҠеҶҚдҪҝз”ЁдёҖеұӮsvnзҡ„еҒҡжі•гҖӮ
